# MarkSpec Examples

Standalone files demonstrating MarkSpec features.

- **requirement-block.md** — entry blocks with typed IDs and attributes
- **in-code-rust.rs** — requirements in Rust doc comments
- **in-code-kotlin.kt** — requirements in Kotlin doc comments
- **glossary.md** — glossary with cross-references
- **traceability-matrix.md** — generated traceability output

The language specification is at
[docs/specification.md](../docs/specification.md).
